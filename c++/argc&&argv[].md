# int main(int argc,char* argv[])参数解释

* 含义

  * 第一个参数**argc**, 为整型，用来统计程序运行时候发送给main函数的命令行参数的个数

  * 第二个参数 **argv[]**，为字符串数组，用来存放指向的字符串参数的指针数组，每一个元素指向一个参数。

    * ```shell
      argv[0]        #指向程序运行的全路径名
      argv[1]        #指向在命令行中执行程序名后的第一个字符串
      argv[2]        #第二个
      argv[3]        #第三个
      argv[argc]     #NULL
      ```
