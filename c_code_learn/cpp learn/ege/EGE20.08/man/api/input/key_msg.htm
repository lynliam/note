主页-＞库函数目录-＞键盘鼠标输入函数-＞key_msg 结构体

功能：
这个结构体用于保存键盘消息

声明：

typedef struct key_msg {
    UINT msg;
    UINT key;
    UINT flags;
}key_msg;



成员：
msg
指定按键消息类型，可为以下值：



值含义


key_msg_down键盘按下消息。


key_msg_up键盘弹起消息。


key_msg_char键盘字符输入消息。




key
如果是按下和弹起的消息，则表示按键虚拟键码，否则为gbk编码字符消息。

flags
按键参数，可能为以下值的组合：



值含义


key_flag_shift同时按下了shift


key_flag_ctrl同时按下了control
