主页-＞库函数目录-＞文字输出相关函数-＞xyprintf

功能：
这个函数用于在指定位置格式化输出字符串。

声明：

void xyprintf(
    int x,
    int y,
    int w,
    int h,
    LPCSTR textstring,
    ...
);
    void xyprintf(
    int x,
    int y,
    int w,
    int h,
    LPCWSTR textstring,
    ...
);



参数：
x, y, w, h
要输出字符串所在的矩形区域

textstring
要输出的字符串的指针。
...
要输出的内容的参数，类似printf。

返回值：
（无）
