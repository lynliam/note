主页-＞库函数目录-＞绘制图形相关函数-＞ege_line

功能：
这个函数用于画线。

声明：

void ege_line(
    float x1,
    float y1,
    float x2,
    float y2,
    PIMAGE pimg = NULL
);



注意事项：
使用的是ARGB颜色，透明度不能为0，否则无绘制效果。

参数：
x1
线的起始点的 x 坐标。

y1
线的起始点的 y 坐标。

x2
线的终止点的 x 坐标（该点本身画不到）。

y2
线的终止点的 y 坐标（该点本身画不到）。

pimg
绘制的目标图像，默认为NULL，即绘制到窗口。

返回值：
（无）
