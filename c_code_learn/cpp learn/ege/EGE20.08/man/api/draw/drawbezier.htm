主页-＞库函数目录-＞绘制图形相关函数-＞drawbezier

功能：
这个函数用于画bezier曲线。边线颜色由setcolor函数决定

声明：

void drawbezier(
    int numpoints,
    const int *polypoints,
    PIMAGE pimg = NULL
);



参数：
numpoints
多边形点的个数，需要是被3除余1的数，如果不是，则忽略最后面若干个点。

polypoints
每个点的坐标（依次两个分别为x,y），数组元素个数为 numpoints * 2。
每一条bezier曲线由两个端点和两个控制点组成，相邻两条则共用端点。

返回值：
（无）
