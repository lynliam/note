主页-＞库函数目录-＞绘制图形相关函数-＞ege_rectangle

功能：
这个函数用于画矩形线框，线框颜色由setcolor指定。

声明：

void ege_rectangle(
    float x,
    float y,
    float w,
    float h,
    PIMAGE pimg = NULL
);



注意事项：
使用的是ARGB颜色，透明度不能为0，否则无绘制效果。

参数：
x
矩形左上角 x 坐标。

y
矩形左上角 y 坐标。

w
矩形宽。

h
矩形高。

pimg
绘制的目标图像，默认为NULL，即绘制到窗口。

返回值：
（无）
