# CSS 样式

## 1.0   CSS 选择器

例子：

```css
/* 元素选择器 */
p {
  text-align: center;
  color: red;
}

/* id 选择器 */
#para1 {
  text-align: center;
  color: red;
}

/* 类选择器 */
.center {
  text-align: center;
  color: red;
}
p.center {
  text-align: center;
  color: red;
}
```



## 2.0    CSS 外部引用

```css
<head>
<link rel="stylesheet" type="text/css" href="mystyle.css">
</head>
```
